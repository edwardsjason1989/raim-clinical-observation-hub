## Overview

Raim provides a web-accessible workspace for handling medical annotations associated with skin changes. It is meant to centralize recording, review, and analysis so dermatology-focused teams can keep observations organized and easier to compare.

The platform supports both doctor and patient usage, with functions aimed at long-term monitoring and structured data handling for diagnostic support. Since it operates in the browser, Raim does not depend on a local desktop installation and can be used from different devices and environments.
